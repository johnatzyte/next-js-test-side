# Simple Next.js Ecommerce Demo

This is a simple e-commerce demo site built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Product Listing Page
- Product Detail Page
- Mock Product Data
- Responsive Design

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
